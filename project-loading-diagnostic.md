# Space Loading Diagnostic

Historical live-loading diagnostic retained for context. The current Hub API
container vocabulary is `spaces`; work areas inside a space are `projects`.

Use current diagnostics against:

- Space list: `/api/hub/spaces`
- Space detail: `/api/hub/spaces/:spaceId`
- Space work projects: `/api/hub/spaces/:spaceId/projects`
- Space members: `/api/hub/spaces/:spaceId/members`
