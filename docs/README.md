# Hub OS — Documentation

Documentation for this project lives in two places:

- **`working files/`** — Active working documents (change log, audits)
- **`apps/hub-api/api-snapshot.json`** — Canonical API route inventory (72 routes)
- **`apps/hub-api/db/`** — Data access layer (statements, transactions, schema, migrations)

Do not create speculative contract documents, design handoffs, or audit reports in this directory unless they are generated directly from the current codebase and verified against running code.
